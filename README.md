# Personal Color 

## Mục đích
Dự án xác định và phân tích màu sắc cá nhân dựa trên màu da, tóc, mắt để đưa ra bảng màu phù hợp.

## Cấu trúc thư mục
- `color_analysis/` : Code backend, xử lý ảnh, trích xuất màu, train model PCA
- `streamlit/` : Code giao diện Streamlit hiển thị kết quả phân tích màu sắc cá nhân
- `docs/` : Báo cáo, slide, khảo sát người dùng
- `README.md` : Mô tả chung và phân công công việc

## Phân công công việc
- Hân :Làm báo cáo  Code chính – xử lý dữ liệu ảnh, trích xuất màu, train model PCA, quản lý repo
- Nhi  code Streamlit – thiết kế giao diện hiển thị kết quả
- Ánh: Tìm data.
- Hiền: Làm slide thuyết trình 

## Hướng dẫn nhanh
- Chạy backend: hướng dẫn xem trong `color_analysis_backend/README_backend.md`
- Chạy giao diện: hướng dẫn xem trong `streamlit/README_streamlit.md`
- Tài liệu tham khảo: xem trong `docs/`
